#  DICOM Viewer

Pythonを用いた、Axial, Coronal, Sagittalの3断面を同時表示可能な医療用画像ビューアー。特別な解析ソフトをインストールすることなく、Python環境のみでDICOM系列データの読込・確認が可能です。

ダウンロードはReleaseページから。

## 主な機能

* **3断面同時表示**: Axial, Coronal, Sagittal面を1つのウィンドウに表示
* **物理スケール補正**: DICOMヘッダからピクセル間隔とスライス厚を取得し、各断面のアスペクト比を自動調整
* **断面同期スライス線**:各画像上に、他の2断面の位置を示すカラーのスライス線を表示
* **Axial表示調整**: 解析上の都合に合わせ、Axial画像を上下反転して表示
* **DICOM情報表示**: 画像サイズ、スライス厚、スライス数をヘッダから抽出して表示


## セットアップと実行手順 (Cloneして利用する場合)

本リポジトリを自身の環境にコピーして動作させるための手順は以下の通りです。

### 1. リポジトリのクローン

ターミナルまたはコマンドプロンプトを開き、以下のコマンドを実行してローカル環境にダウンロードしてください。

```bash
git clone https://github.com/kunikyu/dicom-viewer.git
cd dicom-viewer

```

### 2. 仮想環境の作成 (推奨)

既存のPython環境を汚さないよう、仮想環境を作成して有効化することを推奨します。

* **Windows**:
```bash
python -m venv venv
venv\Scripts\activate
```


* **macOS / Linux**:
```bash
python3 -m venv venv
source venv/bin/activate
```



### 3. 必要ライブラリのインストール

以下のコマンドを実行し、必要なライブラリをインストールしてください
```bash
pip install requirements.txt
```



### 4. アプリケーションの起動

メインスクリプトを実行すると、ビューアーのウィンドウが立ち上がります。

```bash
python DICOM_Viewer.py
```
## 使用方法

`.exe`ファイルの取得と実行方法についてはReleaseページに記載しています。

1. `DICOM_Viewer.py` を実行
2. 「DICOMフォルダを開く」ボタンをクリックし、`.dcm` ファイル群が含まれるディレクトリを選択
3. 読み込み完了後、下部のスライダで各断面のスライス移動、およびWW/WLの調整を行う
4. ウィンドウを閉じるとプログラムは完全に終了する

![ウィンドウ画面](https://scrapbox.io/files/697c07cd6e4528bece702b0e.png)

## 操作・表示仕様

| 項目 | 内容 |
| --- | --- |
| **スライス線（赤）** | Sagittal面の位置 |
| **スライス線（緑）** | Coronal面の位置 |
| **スライス線（水色）** | Axial面の位置 |
| **Axial画像** | 原画像に対して上下反転して表示 |
| **自動初期化** | 読込時、データ分布に基づき最適な明るさに自動設定 |

## 注意事項

* 本ソフトウェアは学習および研究支援用であり、診断目的の医療機器ではありません。
* フォルダ内にDICOMファイル以外の `.dcm` 拡張子を持つファイルが含まれている場合、エラーの原因となる可能性があります。
